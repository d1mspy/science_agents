# Science Agents
проект Science Agents - это продвинутая мультиагентная система на базе Python и LangGraph для автоматизации работы с научными статьями (arXiv)

## Возможности
Система построена на нескольких AI-агентах, каждый из которых выполняет свою роль в обработке научных публикаций.

**Coordinator Agent** отвечает за управление процессом: понимает запрос пользователя, ищет статьи, скачивает PDF, извлекает текст и передает задачи другим агентам. Для этого используются arXiv API, PyMuPDF и Tavily Search.

**Eval Agent** выполняет академическую оценку статьи по критериям ведущих конференций, включая новизну, методологию и потенциальное влияние. Результатом работы является структурированный вердикт Accept или Reject с аргументацией.

**Writer Agent** формирует подробные обзоры научных статей на русском языке. Он выделяет основные идеи, результаты экспериментов, практическую пользу и при необходимости уточняет термины через поиск.

## Технологический стек
- **Python 3.11+**
- **LangGraph / LangChain** — управление мультиагентной логикой
- **OpenRouter / OpenAI** — подключение LLM-моделей
- **arXiv API** — поиск и метаданные статей
- **PyMuPDF4LLM** — извлечение текста из PDF в Markdown
- **Pydantic** — валидация и структурированный вывод

## Установка

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/d1mspy/science_agents
    cd science_agents
    ```

2.  **Создайте виртуальное окружение:**
    ```bash
    python -m venv .venv
    source .venv/bin/activate  # На Windows: .venv\Scripts\activate
    ```

3.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Настройте переменные окружения:**
    Создайте файл `.env` в корне проекта и добавьте ключи API:
    ```env
    OPENROUTER_API_KEY=your_openrouter_key
    OPENROUTER_BASE_URL=https://openrouter.ai/api/v1
    TAVILY_API_KEY=your_tavily_key  # Для поиска в интернете (Writer Agent)
    ```

## Использование

Запустите главного координатора через CLI:

```bash
python science_agents/graph_mas.py
```
